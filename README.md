# 👗 Модный Бутик — Веб-сайт (Контрольная работа)

## 📖 О проекте

**Модный Бутик** — это сайт магазина одежды в российском стиле. Сайт состоит из 3 страниц:

| Страница | Описание |
|----------|----------|
| Главная | Приветствие, акции, преимущества магазина |
| Каталог | Все товары по категориям (платья, юбки, верхняя одежда, обувь) |
| Контакты | Адрес, телефон, форма обратной связи |

## 🛠 Технологии

- HTML5
- CSS3 (адаптивный дизайн, анимации)
- JavaScript (только для уведомлений)

## 📁 Файлы

- `index.html` — главная
- `catalog.html` — каталог
- `contact.html` — контакты
- `style.css` — стили
- `script.js` — скрипты
- `README.md` — описание

## 🚀 Запуск

1. Все файлы положите в одну папку
2. Откройте `index.html` в браузере

## 🎨 Особенности дизайна

- Темная шапка (российский стиль)
- Бежевые акценты
- Карточки товаров с hover-эффектом
- Полностью адаптивно
